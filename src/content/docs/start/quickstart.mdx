:::note
OpenClaw 需要 Node 22 或更新版本。
:::


## 安装

<Tabs>
  <TabItem label="npm">
    ```bash
    npm install -g openclaw@latest
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add -g openclaw@latest
    ```
  </TabItem>
</Tabs>

## 新手引导并运行 Gateway 网关

<Steps>

1. 新手引导并安装服务

   ```bash
       openclaw onboard --install-daemon
       ```

2. 配对 WhatsApp

   ```bash
       openclaw channels login
       ```

3. 启动 Gateway 网关

   ```bash
       openclaw gateway --port 18789
       ```

</Steps>

完成新手引导后，Gateway 网关将通过用户服务运行。你也可以使用 `openclaw gateway` 手动启动。


:::note
之后在 npm 安装和 git 安装之间切换非常简单。安装另一种方式后，运行
`openclaw doctor` 即可更新 Gateway 网关服务入口点。
:::


## 从源码安装（开发）

```bash
git clone https://github.com/openclaw/openclaw.git
cd openclaw
pnpm install
pnpm ui:build # 首次运行时会自动安装 UI 依赖
pnpm build
openclaw onboard --install-daemon
```

如果你还没有全局安装，可以在仓库目录中通过 `pnpm openclaw ...` 运行新手引导。

## 多实例快速开始（可选）

```bash
OPENCLAW_CONFIG_PATH=~/.openclaw/a.json \
OPENCLAW_STATE_DIR=~/.openclaw-a \
openclaw gateway --port 19001
```

## 发送测试消息

需要一个正在运行的 Gateway 网关。

```bash
openclaw message send --target +15555550123 --message "Hello from OpenClaw"
```
